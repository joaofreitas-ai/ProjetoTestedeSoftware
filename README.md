# MovieMatch V3

Projeto Java desenvolvido para a disciplina de Testes de Software.

O sistema simula uma plataforma de recomendação de filmes, utilizando conceitos de Programação Orientada a Objetos, testes unitários e mocks com Mockito.

## Funcionalidades

- Recomendação de filmes
- Filtro de filmes
- Cálculo de score
- Histórico de usuários
- Notificações
- Testes automatizados

## Tecnologias

- Java
- JUnit 5
- Mockito

## Estrutura

- `model` → entidades do sistema
- `service` → regras de negócio
- `exception` → exceções personalizadas
- `util` → utilidades auxiliares
- `test` → testes unitários

- testes de software
